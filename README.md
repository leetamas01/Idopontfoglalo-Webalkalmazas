Időpont foglaló Webalkalmazás

A használt IDE: Visual Studio 2022
A programot a solution (.sln) fájl importálásával lehet betölteni.
A webalkalmazáson bármilyen témáju időpontokat lehet hírdetni és a meghírdetett időpontokat le lehet foglalni.

A webalkalmazás windowson/linuxon is lehet egyaránt futtatni.
